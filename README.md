# Bank Dataset Exploratory Data Analysis (EDA)

## Objective
The goal of this project is to perform exploratory data analysis on a bank
customer dataset to understand data structure, identify patterns, trends,
outliers, and data quality issues.

## Dataset
- Public Bank Customer Dataset
- Source: Open-source / Kaggle

## Analysis Performed
- Data structure and data type exploration
- Missing value analysis
- Frequency tables for categorical variables
- Bar charts and pie charts for visualization
- Identification of trends and patterns
- Detection of potential data quality issues

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Outcome
This analysis provides insights into customer behavior and highlights
important factors that can be used for further statistical analysis
or machine learning tasks.
